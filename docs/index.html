What is an Induction Head?

Induction heads complete patterns by copying what followed previous occurrences

B?

When the model sees [A][B]...[A], an induction head attends back to [B]

Detection Metrics

1 Induction Score

When tokens[i-1] matches tokens[j], measure attention from position i to j+1. High scores mean the head attends to "what came after the previous match."

2 Prefix Matching

Compare attention when prefixes match vs. don't match. Strong induction heads show higher attention when the surrounding context is similar.

3 Copying Score

Measure if attention leads to correct predictions. When max attention points to position j, does tokens[j+1] match what we're predicting?

Quick Start

from induction_head_detector import ( InductionHeadDetector, detect_induction_heads, create_analysis_report, ) # Quick detection result = detect_induction_heads(model, threshold=0.4) print(f"Found {result.num_heads_detected} induction heads") # Detailed analysis detector = InductionHeadDetector(model, threshold=0.3) result = detector.detect(sequence_length=64) for score in result.get_top_heads(n=5): print(f"L{score.layer}H{score.head}: {score.overall_score:.3f}") # Generate report report = create_analysis_report(result, detailed=True) print(report)

Test Pattern Generation

Repeated Sequence

[A,B,C,D,A,B,C,D,A,B,C,D]

Classic pattern for detecting induction. Each repetition provides opportunities for pattern matching.

ABAB Pattern

[A,B,A,B,A,B,A,B]

Simple alternating pattern. Maximum induction signal with frequent repetitions.

Copy Task

[content][SEP][content]

Tests copying across a separator. Similar to in-context learning setups.

Prefix Matching

[...A,B,...,A,B,?]

Tests if the head uses multi-token context for matching, not just single tokens.

Why It Matters

In-Context Learning

Induction heads are the primary mechanism for in-context learning in small to medium models. Understanding them is key to understanding few-shot capabilities.

Phase Transitions

Induction heads emerge suddenly during training - a phase transition. Finding them helps track model capability development.

Circuit Analysis

Induction heads are part of a two-head circuit: previous token heads compose with induction heads. This is a key example of attention composition.

Debugging

If a model fails at pattern completion or copying, checking for induction head development can help diagnose training issues.
